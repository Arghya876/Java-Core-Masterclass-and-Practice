# Phase 4: Java Collections Framework 🗃️

Welcome to **Phase 4**! Learn how Java provides pre-built data structures to store, manage, and search groups of objects effortlessly.

---

## 💡 Key Data Structures
1. **`ArrayList`**: Dynamic auto-resizing array list (Shopping list, student list).
2. **`HashSet`**: Collection of unique items (No duplicates allowed!).
3. **`HashMap`**: Key-Value dictionary (`Key -> Value`) e.g. Phonebook (`Name -> Phone`).

---
