# Phase 2: Object-Oriented Programming (OOPs) Made Easy! 🧩

Welcome to **Phase 2**! In this module, you will learn how real-world objects like Cars, Bank Accounts, Animals, and Shapes are represented in Java code using **Object-Oriented Programming (OOPs)**.

---

## 💡 The 4 Pillars of OOPs (Beginner Friendly Guide)

```
                 +-----------------------------------+
                 |     The 4 Pillars of OOPs         |
                 +-----------------------------------+
                 /          |           |            \
                /           |           |             \
      Encapsulation    Inheritance  Polymorphism   Abstraction
      (Capsule Pill)  (Parent/Child)  (Many Forms) (TV Remote)
```

1. 💊 **Encapsulation**: Bundling variables and hiding data (`private` fields with `getters` and `setters`) so balance/credentials cannot be hacked.
2. 👨‍👦 **Inheritance**: Passing traits from Parent class to Child class using `extends` (saves typing duplicate code!).
3. 🎭 **Polymorphism**: One method name having multiple behaviors (Overloading with different parameters, Overriding in child classes).
4. 📺 **Abstraction**: Hiding internal complex wiring and showing only the simple buttons to press (using `abstract` classes and `interfaces`).
