# Phase 5: File Operations & I/O 📝

Welcome to **Phase 5**! Learn how your Java programs can read and write permanent text files on your hard drive.

---

## 💡 Key Concepts
1. **`File`**: Represents a file path on disk.
2. **`FileWriter`**: Opens a file and writes characters/text lines.
3. **`Scanner`**: Reads text lines sequentially from a file.
