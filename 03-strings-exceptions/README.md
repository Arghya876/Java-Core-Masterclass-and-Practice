# Phase 3: Strings & Exception Handling 🛡️

Welcome to **Phase 3**! Learn how to manipulate text safely and catch program errors before your code crashes.

---

## 💡 Key Concepts Covered
1. **String Manipulation**: `length()`, `charAt()`, `substring()`, `toUpperCase()`, `.equals()`.
2. **String Constant Pool (SCP)**: Memory optimization for identical String literals.
3. **Try-Catch-Finally**:
   - `try`: Put risky code here.
   - `catch`: Catch specific errors (`ArithmeticException`, `ArrayIndexOutOfBoundsException`).
   - `finally`: Code that ALWAYS runs (closing files, releasing memory).
