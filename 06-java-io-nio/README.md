# Phase 6: Java I/O & NIO.2 Roadmap

Welcome to **Phase 6** of the Java Core Roadmap! Learn to read and write data to files, streams, buffers, and network channels efficiently.

---

## 🎯 Core Learning Objectives

### 1. Traditional Java I/O (`java.io.*`)
- Byte Streams (`InputStream`, `OutputStream`, `FileInputStream`, `FileOutputStream`).
- Character Streams (`Reader`, `Writer`, `FileReader`, `FileWriter`).
- Buffered Streams (`BufferedReader`, `BufferedWriter`, `BufferedInputStream`).
- Object Serialization (`Serializable` interface, `transient` keyword, `ObjectInputStream`/`ObjectOutputStream`).

### 2. Java NIO.2 (`java.nio.file.*`)
- `Path`, `Paths`, and `Files` utilities.
- High-performance non-blocking file reads and writes (`Files.readAllLines`, `Files.walk`).
- Memory-mapped files and channels.
