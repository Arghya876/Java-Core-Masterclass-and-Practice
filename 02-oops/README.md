# Phase 2: Object-Oriented Programming (OOPs) Roadmap

Welcome to **Phase 2** of the Java Core Roadmap! Having mastered Java Basics (syntax, data types, control flow, arrays, math logic, and methods), Phase 2 transitions from procedural programming to **Object-Oriented Programming**.

---

## 🎯 Core Learning Objectives

### 1. Classes & Objects
- Understanding Class as a Blueprint vs Object as an Instance.
- Fields (State) & Methods (Behavior).
- Memory allocation on Heap (`new` keyword) & Stack (reference variables).
- Constructors (Default, Parameterized, Copy Constructor).
- Keyword `this`: Variable shadow resolution and constructor chaining.

### 2. The Four Pillars of OOPs
1. **Encapsulation**:
   - Access Modifiers (`private`, `default`, `protected`, `public`).
   - Getter and Setter methods with validation.
   - Data Hiding & Read-Only / Write-Only properties.
2. **Inheritance**:
   - `extends` keyword. Single, Multilevel, and Hierarchical Inheritance.
   - Why Multiple Inheritance is disallowed for classes (Diamond Problem).
   - Keyword `super`: Calling parent constructor and overridden methods.
3. **Polymorphism**:
   - **Compile-Time Polymorphism (Static Binding)**: Method Overloading.
   - **Runtime Polymorphism (Dynamic Binding)**: Method Overriding (`@Override`).
   - Dynamic Method Dispatch (Upcasting & Covariant Return Types).
4. **Abstraction**:
   - Abstract Classes (`abstract` keyword, abstract vs concrete methods).
   - Interfaces (`interface` keyword, `implements`).
   - Multiple Inheritance via Interfaces.
   - Java 8+ Interface features: `default` and `static` methods.
   - Java 9+ `private` interface methods.

### 3. Advanced Class Concepts
- Static Members (`static` variables, `static` methods, `static` initializer blocks).
- Final Keyword (`final` variables, `final` methods, `final` classes).
- Inner Classes (Member Inner, Static Nested, Local Inner, Anonymous Inner Classes).
