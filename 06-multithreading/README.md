# Phase 6: Multithreading & Concurrency ⚡

Welcome to **Phase 6**! Learn how to run multiple tasks at the exact same time using Java Threads.

---

## 💡 Key Concepts
1. **Thread**: A lightweight thread of execution inside JVM.
2. **`Runnable` Interface**: Represents a task that can be executed by a thread (`run()` method).
3. **`Thread.sleep(milliseconds)`**: Pauses execution for a specified duration.
