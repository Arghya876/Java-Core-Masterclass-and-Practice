## 📌 3. Quick Beginner Q&A

### Q1: What does `public static void main(String[] args)` mean?
- `public`: Accessible from anywhere.
- `static`: Can run directly without creating a new object first.
- `void`: Does not return any value back after finishing.
- `main`: The entry point name the JVM searches for to start your program.
- `String[] args`: Command-line options passed when launching the program.

### Q2: What is the difference between `System.out.print()` and `System.out.println()`?
- `System.out.print()`: Prints text and keeps the cursor on the **same line**.
- `System.out.println()`: Prints text and moves the cursor to a **new line** below.

### Q3: Why does `10 / 3` equal `3` instead of `3.333` in Java?
Because both `10` and `3` are integers (`int`). Java cuts off the decimal portion!
To get floating-point results like `3.333`, write `10.0 / 3.0`.
