# Step 9: Methods, Functions & Recursion 🛠️

Welcome to **Step 9**! **Methods** (also called functions) allow you to write reusable blocks of code, break large programs into clean modules, and write recursive solutions.

---

## 💡 Key Concepts Covered
1. Method Anatomy: `accessModifier static returnType methodName(parameters)`.
2. Parameters vs Arguments.
3. Return Types (`void` vs `int`/`double`/`String`).
4. **Method Overloading**: Same method name with different parameter lists.
5. **Pass-by-Value**: Understanding how primitive copies vs object reference copies behave.
