# Step 8: Math & Number Logic Practice Programs 🔢

Welcome to **Step 8**! This folder contains classic logical & mathematical interview programs designed to build strong problem-solving skills.

---

## 💡 Topics & Code Files

| File Name | Problem Concept & Description |
| :--- | :--- |
| [`CountDigits.java`](./CountDigits.java) | Count total digits in an integer (`while(n > 0) n /= 10`). |
| [`SumDigit.java`](./SumDigit.java) | Calculate sum of all digits in a number. |
| [`RevNum.java`](./RevNum.java) | Reverse integer digits (e.g. `1234` -> `4321`). |
| [`Factorial.java`](./Factorial.java) | Factorial of $N$ ($N! = N \times (N-1) \dots 1$). |
| [`Power.java`](./Power.java) | Calculate $base^{exponent}$ using loops. |
| [`Prime.java`](./Prime.java) | Optimized prime number check up to $\sqrt{N}$. |
| [`PrimeNumBetweenTwo.java`](./PrimeNumBetweenTwo.java) | Generate all prime numbers within an interval $[A, B]$. |
| [`PalindromeNum.java`](./PalindromeNum.java) | Check if a number reads same backwards (`12321`). |
| [`ArmstrongNum.java`](./ArmstrongNum.java) | Narcissistic number check ($153 = 1^3 + 5^3 + 3^3$). |
| [`PerfectNum.java`](./PerfectNum.java) | Check if sum of proper divisors equals the number (`6 = 1+2+3`). |
