# 🟢 Module 01: Java Basics & Fundamentals

Welcome to **Module 01: Java Basics** of the **Java Learning Concepts** repository.

This module covers foundational core Java concepts, syntax, control structures, arrays, math algorithms, and methods.

---

## 📌 Module Contents & Topics

### 1. Introduction & Syntax
- [HelloWorldAndJVM.java](./HelloWorldAndJVM.java): Main entry point, structure, and JVM overview.
- [Literals.java](./Literals.java): Integer, floating-point, character, boolean, and String literals.
- [ASCII.java](./ASCII.java): Character encoding and ASCII values in Java.

### 2. Variables & Data Types
- [DataTypesAndCasting.java](./DataTypesAndCasting.java): Primitive data types, size, implicit/explicit type casting.
- [Average.java](./Average.java): Basic arithmetic operation and average calculation.

### 3. User Input & Operators
- [ScannerAndInput.java](./ScannerAndInput.java): Reading console input using `java.util.Scanner`.
- [OperatorsDemo.java](./OperatorsDemo.java): Arithmetic, Relational, Logical, Bitwise, Ternary, Assignment, and Unary operators.

### 4. Decision Making & Conditionals
- [OddEven.java](./OddEven.java): Simple `if-else` condition.
- [MaxNum.java](./MaxNum.java): Finding maximum between two numbers.
- [MaxInThree.java](./MaxInThree.java): Nested `if-else` for maximum among three numbers.
- [VowelConsonant.java](./VowelConsonant.java): `switch-case` statement.
- [TaxCal.java](./TaxCal.java): Income tax slab logic with multiple conditions.

### 5. Loops & Iterations
- [NaturalNum.java](./NaturalNum.java): `while` loop for printing natural numbers.
- [CharPrint.java](./CharPrint.java): Iterating character ranges with loops.
- [MulTable.java](./MulTable.java): Multiplication table generator.
- [SumNNum.java](./SumNNum.java): Sum of first N natural numbers.
- [OddNumInRange.java](./OddNumInRange.java): Range filtering with loops.

### 6. Arrays & Data Structures
- [ArrayDemo.java](./ArrayDemo.java): 1D Array declaration, allocation, indexing.
- [ArrayMethod.java](./ArrayMethod.java): Array manipulation methods.
- [SumArrayEle.java](./SumArrayEle.java): Summing array elements.
- [AveArray.java](./AveArray.java): Calculating mean of array values.
- [BiggestEleArray.java](./BiggestEleArray.java) & [SmallestEleArray.java](./SmallestEleArray.java): Min/Max search in arrays.
- [ArrayRev.java](./ArrayRev.java): In-place array reversal.
- [CopyArrEle.java](./CopyArrEle.java): Copying array contents.
- [ScearchEleInArr.java](./ScearchEleInArr.java): Linear Search algorithm.
- [BinarySearchDemo.java](./BinarySearchDemo.java): Binary Search implementation.
- [SortArrayInAscending.java](./SortArrayInAscending.java) & [SortArrayInDescending.java](./SortArrayInDescending.java): Array sorting algorithms.
- [MatrixOperations.java](./MatrixOperations.java): 2D Arrays, Matrix addition, subtraction, multiplication.

### 7. Math & Logical Programs
- [CountDigits.java](./CountDigits.java) & [SumDigit.java](./SumDigit.java): Digit extraction and summation.
- [RevNum.java](./RevNum.java) & [PalindromeNum.java](./PalindromeNum.java): Reversing numbers and palindrome check.
- [Factorial.java](./Factorial.java): Iterative factorial calculation.
- [Power.java](./Power.java): Exponential power calculation (`a^b`).
- [Prime.java](./Prime.java) & [PrimeNumBetweenTwo.java](./PrimeNumBetweenTwo.java): Prime number algorithms.
- [FibonacciSeries.java](./FibonacciSeries.java): Fibonacci series generation.
- [GCDandLCM.java](./GCDandLCM.java): Euclidean algorithm for GCD and LCM.
- [ArmstrongNum.java](./ArmstrongNum.java): Armstrong number check.
- [PerfectNum.java](./PerfectNum.java): Perfect number determination.

### 8. Methods & Recursion
- [MethodBasics.java](./MethodBasics.java): Method declaration, parameters, return types, overloading.
- [PassByValueDemo.java](./PassByValueDemo.java): Pass-by-value semantics in Java.

---
