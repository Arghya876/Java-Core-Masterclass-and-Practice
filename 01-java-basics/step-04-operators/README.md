# Step 4: Operators & Expressions ➕➖✖️➗

Welcome to **Step 4**! Operators allow Java to perform calculations, compare values, evaluate logical conditions, and manipulate binary bits.

---

## 💡 Types of Operators in Java

1. **Arithmetic**: `+`, `-`, `*`, `/`, `%` (Modulus / Remainder).
2. **Increment & Decrement**: `x++` (Postfix), `++x` (Prefix).
3. **Relational**: `==`, `!=`, `>`, `<`, `>=`, `<=`.
4. **Logical**: `&&` (AND), `||` (OR), `!` (NOT).
5. **Bitwise**: `&`, `|`, `^`, `~`, `<<`, `>>`.
6. **Ternary**: `condition ? value1 : value2`.

---

## 🧠 Arithmetic Modulus (`%`) Explanation
The modulus operator `%` returns the **remainder** of division:
- `10 / 3` = `3` (Quotient)
- `10 % 3` = `1` (Remainder because $3 \times 3 = 9$, leftover is $1$)

Modulus is heavily used to check if a number is **Even/Odd** (`num % 2 == 0`) or extract the last digit of a number (`num % 10`).
