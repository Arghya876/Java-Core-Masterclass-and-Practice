# Step 5: Decision Making & Conditionals 🚦

Welcome to **Step 5**! Conditionals allow your program to make decisions and execute different blocks of code based on true/false conditions.

---

## 💡 Key Structures Covered
- `if (condition)`: Executes code if condition is `true`.
- `if-else`: Chooses between two paths.
- `if-else if-else`: Evaluates multiple conditions sequentially.
- `switch-case`: Efficient branch selection based on a single value.

---

## 🧠 Simple Analogy: Traffic Lights 🚦
```java
if (light.equals("RED")) {
    System.out.println("STOP!");
} else if (light.equals("YELLOW")) {
    System.out.println("SLOW DOWN!");
} else {
    System.out.println("GO!");
}
```
